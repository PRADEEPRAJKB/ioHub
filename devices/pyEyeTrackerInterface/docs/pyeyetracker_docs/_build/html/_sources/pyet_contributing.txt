*****************
Want to Help Out?
*****************

If you would like to contribute to the pyEyeTracker project, we could likely use it. ;) Here are just some of the areas
that assistance could be provided in:

*providing input on an area of eye tracking research or type of eye tracker that you think the pyEyeTracker specification is currently missing
*helping with documentation
*improving the specifications design or functionality implementation wise
*implemetation of a specific eye tracker model integration
*helping other users by answering questions they have posted
*testing
*funding and awareness oppertunities
*.....

If you would like to help, please do by contacting us so we can welcome you and work with 
you to determine how you could best help the project.

**Thank you for you interest!**
